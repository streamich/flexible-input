# Flexible `<input>` and `<textarea>` elements

This package implements flexible `<input>` and `<textarea>` which expand and
shrink to fit the content.


## Usage

Installation:

```
npm install flexible-input
```

Usage:

```tsx
import {FlexibleInput} from 'flexible-input';

<FlexibleInput
  value={value}
  onChange={onChange}
  typeahead={'...'}
/>
```
